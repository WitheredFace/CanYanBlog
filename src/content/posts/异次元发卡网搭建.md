# 第一步 准备域名

首先需要一个 **域名**（也就是访问网站的网址）。

推荐使用 **Spaceship**，价格比较便宜

官网：https://www.spaceship.com

购买完成后记住 **域名管理后台**，后面需要在里面添加 **DNS解析**。

---

# 第二步 购买主机

网站需要运行在服务器上。

推荐 **云服务器**，性能更好；如果只是学习或小规模使用，**虚拟主机也可以**。

虚拟主机推荐：https://sadidc.com

购买完成后一般会提供：

- 服务器 **IP地址**
- **数据库信息**
- **主机管理面板**

本教程示例 IP：

**103.43.8.200**

> 提示：这里使用的是示例 IP，实际需要填写 **你自己的服务器或虚拟主机提供的 IP**。

---

# 第三步 安装发卡网

## 下载源码

前往官网下载安装包：

https://faka.wiki

下载 **最新版源码压缩包**。

---

## 上传源码

进入虚拟主机后台 **文件管理**：

1. 上传源码压缩包  
2. 解压到 **网站根目录**

解压后通常会出现一个文件夹。

接下来：

3. 进入这个文件夹  
4. **全选里面所有文件并剪切**  
5. 回到网站根目录，再 **粘贴**

这样可以确保 **所有网站文件都在根目录**，否则网站可能无法正常运行。

---

## 设置环境

在主机后台找到 **PHP版本设置**。

推荐：

- PHP = 8.0  
- MySQL => 5.7

一般虚拟主机的MySQL版本默认已经满足。

---

## 伪静态设置

网站需要配置伪静态，否则部分页面可能打不开。

添加以下规则：

```nginx
location / {
    if (!-e $request_filename){
        rewrite ^(.*)$ /index.php?s=$1 last; break;
    }
}
```

一般在 **伪静态 / Rewrite设置** 中填写。

---

# 第四步 绑定域名

示例域名：

test.xoxor.top

---

## 设置DNS解析

在域名后台添加解析：

| 类型 | 名称 | 值 |
|----|----|----|
| A | test | 103.43.8.200 |

解析完成后访问：

test.xoxor.top

即可打开网站。

> 提示：IP **103.43.8.200 只是示例**，实际需要填写你的服务器 IP。

> [!NOTE]
> 如果服务器IP使用 **IPv6**，解析类型需要选择 **AAAA**。

---

# 第五步 网站安装

访问域名后会进入 **安装界面**。

按照提示填写数据库信息即可。

> [!NOTE]
> 提示：很多虚拟主机 **数据库账号 = 数据库库名**。

安装完成后即可进入后台。

---

# 第六步 基础设置

进入后台后需要修改一个重要设置：

**CDN获取IP方式选择：`HTTP_X_FORWARDED_FOR`**

如果不设置，可能会出现：

- 登录状态几分钟就失效  
- 后台提示登录过期

---

# 第七步 分站配置

如果需要开启分站功能：

进入后台：

`后台` → `其他设置`

---

## 主站域名

填写你的顶级域名，例如：

xoxor.top

如果用户创建分站，地址会变成：

user.xoxor.top

---

## DNS-CNAME

填写主站地址：

test.xoxor.top

这个主要用于 **分站绑定独立域名**。

---

## 泛解析设置

在域名后台新增解析：

| 类型 | 名称 | 值 |
|----|----|----|
| A | * | 103.43.8.200 |

效果：

*.xoxor.top

例如：

user1.xoxor.top  
user2.xoxor.top

都可以访问。

最后在虚拟主机后台 **再绑定一次域名**：

```
*.xoxor.top
```

---

# 第八步 配置支付

进入后台：

`支付管理` → `支付插件`

点击：

`安装更多插件`

安装 **易支付插件**。

---

## 易支付配置

进入：

`支付插件` → `配置`

填写易支付提供的接口信息。

推荐平台：

https://pay.myzfw.com

---

## 启用支付

进入：

`支付管理` → `支付接口`

配置：

- 微信支付  
- 支付宝支付  

按照提供的信息填写即可。
